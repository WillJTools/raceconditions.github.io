CANNON

Race Condition • Evidence Mode • Python Generator

Paste a Burp raw request or a curl command. Cannon generates a Python evidence script that fires synchronized bursts (max 10 threads) and produces report-ready artifacts (JSONL/CSV/Summary). Use only on systems you’re authorized to test.

1) Paste Request

Burp raw HTTP or curl
Raw request example: POST /redeem HTTP/2 + headers + blank line + JSON body. Curl example: curl -X POST https://target/api -H 'Content-Type: application/json' -d '{...}'.
Tip: For best “definitive” proof, choose a state-changing endpoint (redeem/apply/claim/checkout) and set a success condition that reflects a real outcome.
